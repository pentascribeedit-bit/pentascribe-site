Validar certificado / Validate certificate

Insira o código de validação informado no certificado. A validação confirma a existência do código no registro público e exibe metadados essenciais, preservando a privacidade do autor/solicitante e do conteúdo avaliado.

Use the certificate’s validation code to confirm authenticity. The code is verified against the public registry, and key metadata are returned, while the author/requester’s privacy and the document content are protected.

Dica: o certificado pode conter um QR code que abre esta validação com o código já preenchido. / Tip: the certificate may include a QR code that opens this page with the code prefilled.

Como a validação funciona / How validation works

A PentaScribe registra um código único por certificado emitido. Ao inserir o código, o sistema retorna o status do registro e informações mínimas para confirmação (tipo de serviço, padrão linguístico e data). Certificados revogados aparecem com status “Revogado”.

Each certificate has a unique code. When you enter it, the system verifies the record and returns the status and key metadata (service type, language standard, and issue date). Revoked certificates are flagged as “Revoked.”

Privacidade / Privacy

Este verificador não exibe dados pessoais além do que constar no próprio certificado e não divulga conteúdo do manuscrito. O objetivo é permitir validação pública sem exposição do material científico.

Validation shows only certificate metadata. It does not disclose additional personal data or any manuscript content, enabling public verification without exposing the scientific material.
