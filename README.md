# PentaScribe — Site multi-páginas (gratuito) com simulador

Arquivos:
- index.html (Início)
- servicos.html
- planos.html
- simulador.html (Simular orçamento)
- faq.html
- contato.html
- styles.css
- script.js

## Publicar gratuitamente (GitHub Pages)

1) Crie um repositório no GitHub (ex.: `pentascribe-site`).
2) Faça upload de todos os arquivos acima (na raiz do repositório).
3) Em **Settings → Pages**:
   - Source: Deploy from a branch
   - Branch: main / (root)
4) O GitHub mostrará a URL do site.

## Editar valores do simulador

Abra `script.js` e altere `PRICING` (tarifas e prazos).

## Observação

Este site é estático (sem backend) e não coleta dados.
Para formulário com envio automático, você pode integrar um serviço externo (ex.: Formspree) ou incorporar um Google Form.
